## SimuladorGrafico_ED

Aplicação gráfica das seguintes estruturas de dados, estudadas ao longo da disciplina:

- Listas Sequenciais
- Listas Encadeadas
- Filas
- Pilhas
- Árvores Binárias de Pesquisa

Com a implementação de operações como inserção, remoção e busca.
